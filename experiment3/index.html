Experiment 3 - Alternate Worlds

Description

In this experiment a dungeon was generated and a room with various biomes around it was generated

Technical

I used a tilemap to create tiles, and I had an array storing characters that represented each type of tile

Reflection

The process was somewhat hard. For starters, I got messed up because the tilemap is accessed using coordinates responding to x, y, but the data array is accessed using y, x. Understanding what drawcontext did was somewhat confusing, and getting the fog to work the way I wanted was a little difficult. Also, fixing issues with my biome generation took some time.
