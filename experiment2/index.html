Experiment 2 - Living Impressions

Description

In this experiment I am generatively creating a replication of an image with a few differences upon each use of the reimagine button

Technical

I generated trees, clouds, and rocks using random generation using p5.js, and this was done using the shape functoins in p5js like ellipse(). Random generation with a seed ensures the position of everything is random

Reflection

I first found an image I liked and then I started replicating the trees. Then I replicating the clouds and lastly I replicated the rocks. The only difficult part was aligning the trees properly
